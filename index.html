Binary voting machine

Reach group decisions through iterative voting

  1. Set your range (start/end values)
  2. Choose iterations: Each iteration cuts the range in half. Example: Range 0-100 with 4 iterations goes 100→50→25→12→6. Want precision of 10? Use 4 iterations.
  3. Each round: poll your group, press YES if majority (>50%), NO if minority (<50%), I'M NOT SURE if tied (=50%)
  4. Final range = group consensus
